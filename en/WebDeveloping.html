Web Development

How to best connect with people and present yourself? One of the options is social media. I have always liked and been drawn to the idea of having my own website—a place where people can find information they are interested in. So I decided to start creating it. Piece by piece, I combine HTML with CSS. Gradually, I encounter challenges that enrich both the HTML and CSS structure and give me new experience.

Time passes and the project grows with new subpages, and the number of lines of CSS increases. Every section, every image, every new feature adds something. JavaScript and a simple backend are gradually added as needed.

Thanks to this project, I gained experience with:

  • Creating website structure with HTML
  • Styling the website, laying out elements, and working with responsive design (CSS)
  • Using image and video banners
  • Managing basic interactivity (click reactions, form processing)
  • Collecting data from forms and sending it further
  • Understanding how dynamic website behavior works
  • The principle of communication between frontend and backend
