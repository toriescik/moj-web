Tvorba webu

Ako sa najlepšie spojiť s ľudmi a odprezentovať sa? Jednou z možností sú sociálne siete. Vždy sa mi však páčila a lákala myšlienka vlastného webu. Kúsok miesta, kde môže človek nájsť informácie, ktorého by zaujímali. Tak som sa rozhodol a začal tvoriť.Kúsok po kúsku dávam dokopy HTML s CSS. Postupne sa stretávam s výzvami, ktoré obohacujú HTML a CSS štruktúru a zároveń aj mňa o nové skúsenosti.

pohľad na css

Čas plynie a projekt sa rozrastá o nové podstránky, počet riadkov kódu v css pribúda. Každá sekcia, každý obrázok, každá nová funkcia niečo pridá. Pre potreby projektu postupne pribudá JavaScript a jednoduchší backend.

JavaScript LightBox
JavaScript RollingMenu
server.js pre backend

Vďaka projektu mám skúsenosť s :

  • vytváraním štruktúry webovej stránky HTML
  • stránku štýlovať, rozložiť prvky, pracovať s responzívnym dizajnom (CSS)
  • používanie obrazkových a video banenrov
  • ovládanie základnej interaktivity stránky(reakcie na kliknutia, spracovanie formulárov)
  • zbierať údaje z formulárov a odosielať ich ďalej
  • prehľad o tom, ako funguje dynamické správanie webu
  • princíp komunikácie medzi frontendom a backendom
